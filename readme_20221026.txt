# 眼底画像から眼底年齢を予測するAIモデルとサンプルコード

本モデルは学習済み重みパラメータ(.pth形式)で公開しています。
本モデルはImageNet-22k,1kでpre-trainingされた384x384画素を入力するSwin-Transformerモデル*を転移学習して作成されました。
サンプルコードはPython3.8, Pytorch使用で動作します。Windows10上で動作確認しています。

*https://arxiv.org/abs/2103.14030

# 入力画像について

眼底画像の黒余白は切り取り、384x384のサイズにして下さい。
左眼の画像は左右反転して下さい（眼底画像は黄斑と視神経乳頭を含み、視神経乳頭が右側になるようにして下さい）。
[test1.jpg]の画像を参照して下さい。
